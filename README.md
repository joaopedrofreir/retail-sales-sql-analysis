# Análise de Vendas de Supermercado com SQL

## 📊 Sobre o projeto

Este projeto realiza uma análise exploratória de um dataset de vendas de supermercado utilizando SQL no PostgreSQL.  
O objetivo é responder perguntas de negócio a partir dos dados, demonstrando habilidades iniciais em análise de dados.

## 🛠 Tecnologias utilizadas

- PostgreSQL
- SQL
- Git e GitHub

## 📂 Estrutura do projeto

- `SuperMarket Analysis.csv` → dataset utilizado
- `supermarket_sales_analysis.sql` → consultas SQL do projeto

## 🔎 Perguntas analisadas

1. Qual é a margem de lucro geral da empresa?
2. Qual filial possui maior faturamento e lucro?
3. Qual linha de produto gera mais receita?
4. Existe diferença de gasto médio entre gêneros?
5. Qual período do dia gera mais receita?
6. Maior volume de vendas gera maior lucro?
7. Existe padrão de vendas ao longo da semana?
8. O programa de fidelidade impacta o ticket médio?

## 🎯 Objetivo

Demonstrar habilidades em:

- Manipulação de dados com SQL
- Transformação de dados
- Análise exploratória
- Responder perguntas de negócio com dados

## 📈 Principais insights

- A filial **Giza** apresentou o maior faturamento e lucro total.
- O período da tarde gerou a maior receita.
- Clientes do tipo **Member** possuem ticket médio maior e geram mais receita que clientes **Normal**.
